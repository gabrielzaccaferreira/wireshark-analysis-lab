
# ICMP Analysis

## Objetivo

Analisar o funcionamento do protocolo ICMP através do comando ping.

## Filtro Utilizado

icmp

## Teste Executado

```bash
ping -c 4 8.8.8.8
```

## O que foi observado

* Echo Request
* Echo Reply
* Tempo de resposta (RTT)

## Aprendizados

O ICMP é utilizado para diagnóstico de rede e verificação de conectividade entre hosts.
