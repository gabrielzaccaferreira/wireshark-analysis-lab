
# DNS Analysis

## Objetivo

Analisar consultas DNS utilizando Wireshark.

## Filtro Utilizado

dns

## Teste Executado

Consulta ao domínio google.com utilizando:

```bash
nslookup google.com
```

## Resultado

Foi possível identificar:

* Consulta DNS
* Resposta DNS
* Endereços IPv4
* Endereços IPv6

## Aprendizados

* Funcionamento do DNS
* Diferença entre Query e Response
* Resolução de nomes
